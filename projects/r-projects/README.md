# R Data Analysis Projects

Collection of statistical analysis and machine learning projects demonstrating analytical expertise and business impact.

## Projects

### 1. Operational Risk Briefing: Strategic Audit Prioritization
**View:** [rpubs.com/slammerjammer/1422063](https://rpubs.com/slammerjammer/1422063)

**Objective:** Develop predictive model to identify operational risk signatures and optimize audit prioritization

**Technical Approach:**
- Built Random Forest classification model using R (caret package)
- Performed exploratory data analysis to identify risk patterns
- Engineered features from historical operational data
- Validated model performance and identified key risk indicators

**Business Impact:**
- Identified key characteristics of high-risk operations
- Enabled data-driven audit scheduling strategy
- Improved risk assessment accuracy

**Skills Demonstrated:**
- Machine Learning (Random Forest)
- Predictive Modeling & Classification
- Feature Engineering
- Data Visualization (ggplot2)
- Statistical Analysis

**Tools:** R, tidyverse, caret, ggplot2, RMarkdown

### 3. Customer Segmentation Analysis: Cyclistic
**View:** [rpubs.com/slammerjammer/1371255](https://rpubs.com/slammerjammer/1371255)

**Objective:** Understand behavioral differences between customer segments to inform marketing strategy

**Technical Approach:**
- Analyzed 5.6M+ customer records across usage patterns and engagement metrics
- Performed segmentation analysis (casual vs. annual customers)
- Conducted statistical hypothesis testing comparing segments
- Created data visualizations for stakeholder presentation
- Developed data-driven recommendations

**Key Findings:**
- Identified distinct behavioral patterns between customer segments
- Quantified usage differences and engagement metrics
- Provided evidence-based conversion recommendations

**Business Impact:**
- Enabled targeted marketing strategy based on customer behavior
- Informed customer conversion initiatives
- Supported strategic business decisions with data

**Skills Demonstrated:**
- Exploratory Data Analysis (EDA)
- Customer Segmentation
- Statistical Testing & Hypothesis Testing
- Data Visualization (ggplot2)
- Business Analytics
- Stakeholder Communication

**Tools:** R, dplyr, tidyverse, ggplot2, RMarkdown
